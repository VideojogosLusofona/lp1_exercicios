## Problema

20 -  Indica exatamente o que é mostrado no ecrã pelo seguinte código C#,
incluindo espaços. Justifica a tua resposta.

```cs
double d = 0.3513;
float f = 12.645f;

Console.WriteLine("d -> {0,5:f2}", d);
Console.WriteLine("f -> {0,5:f2}", f);
```

## Soluções

d ->  0.35
