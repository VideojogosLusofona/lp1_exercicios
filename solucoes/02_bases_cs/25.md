  
## Problema

25 - Reescreve o seguinte código sem usar `breaks` e `continues`.

```cs
const int maxTurns = 20;
int turn = 0;
while (true)
{
    turn++;

    if (turn >= maxTurns)
    {
        break;
    }

    if (turn % 3 == 0)
    {
        continue;
    }

    Console.WriteLine($"Turn number {turn}");
}

Console.WriteLine($"Last turn was {turn}");
```

## Soluções

### Solução 1

```cs
// Varibles
const int maxTurns = 20;
int turn = 0;

// While true 
while (turn < maxTurns)
{
	turn++;

	if (turn % 3 != 0)
	{
		Console.WriteLine($" Turn number {turn}");
	}
}
Console.WriteLine($" Last turn was {turn} ");
```
