## Pergunta

4 - Para que serve o ficheiro `.gitattributes`?

## Soluções

## Solução 1

O ficheiro ".gitattributes" representa as configurações do Git para o projeto.
Algumas dessas configurações também podem ser especificadas para um determinado
caminho (ou path), para que o Git aplique as configurações apenas para um 
subdiretório de arquivos. Os arquivos ".gitattributes" e ".gitignore" são gerados
automaticamente ao criar um repositório.
