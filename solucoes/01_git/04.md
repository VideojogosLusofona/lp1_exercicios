## Pergunta

4 - Para que serve o ficheiro `.gitattributes`?

## Solução 1
O ficheiro `.gitattributes` representa configurações para determinados tipos de 
arquivo. No `.gitattributes` pode-se definir behaviours e mesclar arquivos com 
qualquer outro arquivo que não esteja em formato de texto (que é considerado um 
arquivo binário).

# Exemplo da solução 1:
Um dos exemplos de configuração para o `.gitattributes` é dizer ao Git para
tratar um determinado arquivo com um determinado atributo, por exemplo com tipos
de imagens, que por padrão são tratados como binários:

*.jpg    binary
*.png    binary
*.gif    binary

# Observações: 
Os arquivos `.gitattributes` e `.gitignore` não são gerados automaticamente ao 
criar um repositório, mas podem ser criados automaticamente pelo Visual Studio 
caso opte por criar um repositório Git quando começar um novo projeto.
